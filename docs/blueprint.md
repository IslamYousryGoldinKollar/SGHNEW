# **App Name**: Trivia Titans

## Core Features:

- Team Creation: Allow users to create or join a team by entering a team name; limit teams to a maximum of 10 players.
- Live Trivia Session: Initiate a live trivia session consisting of 10 pre-defined questions; a timer starts automatically upon session start and lasts for 5 minutes.
- Question Delivery: Deliver questions to each team member in both teams non simultaneously
- Real-time Scoring: Update the scoreboards for both teams in real-time as questions are answered correctly; scores should only be visible to the team the users are playing on.
- Answer Validation: Determine if the user response to the displayed question is correct, then update that team's score. After he answer question, show the correct answer and update the score before displaying the next question to him.
- AI Content Curator: Employ a generative AI tool that assesses questions in terms of difficulty, topic coverage, and overall player engagement, incorporating questions according to these metrics.
- Results Display: Announce the winning team and the score tally for each team. Add an optional celebratory message to the winners.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to convey trustworthiness and engagement.
- Background color: Light blue (#E3F2FD), creating a clean and unobtrusive backdrop.
- Accent color: Vivid purple (#9C27B0) to highlight interactive elements and important notifications, guiding user interaction.
- Body and headline font: 'Space Grotesk', a sans-serif font for clear readability and a modern feel.
- Use sharp and distinct icons for interactive elements and team representation, maintaining a crisp and professional aesthetic.
- Implement a structured layout with dedicated panels for questions, team scores, and timer, providing essential information.
- Incorporate subtle transitions and animations on correct answers and scoreboard updates to provide a polished user experience.
